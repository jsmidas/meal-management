# 조리지시서 - 식수 변동 시 식자재 조정 기능

## 배경
- 발주서 생성 후 식수(인원)가 변경되면, 식자재 수량도 조정이 필요
- 현재는 "추가발주" 모달에서 전체 변동분을 식자재별로 보여주고 있음
- **문제:** 수백 개의 식자재가 한꺼번에 나와서 실무 대응이 어려움

---

## 선택지

### A. 변동 사업장별 선택
- 식수가 변동된 사업장 목록을 먼저 보여줌
- 사업장을 선택하면 해당 사업장에 해당하는 식자재만 자동 계산
- **장점:** 특정 거래처 단위로 대응 가능
- **단점:** 같은 식자재가 여러 사업장에 걸칠 수 있음

### B. 변동 메뉴별 선택
- 식수 변동으로 영향받는 메뉴(예: 간장불고기, 백미밥) 목록을 보여줌
- 메뉴를 선택하면 해당 식자재만 차이분 계산
- **장점:** 조리 단위로 직관적
- **단점:** 메뉴 수가 많으면 여전히 복잡

### C. 변동분 자동 전체 반영
- 식수 증감분에 해당하는 모든 식자재를 자동 계산해서 한번에 보여줌
- 체크박스로 개별 제외 가능
- **장점:** 빠르고 간단, 놓치는 항목 없음
- **단점:** 항목이 많을 수 있음 (현재 방식과 유사하나 그룹핑 개선)

### D. 간단 조정 패널
- 경고 배너에서 바로 '변동분 반영' 버튼을 누르면
- 증가/감소된 식자재만 요약 팝업으로 보여주고
- 확인 후 추가발주 생성
- **장점:** 최소 클릭으로 빠른 대응
- **단점:** 세부 조정이 제한적
